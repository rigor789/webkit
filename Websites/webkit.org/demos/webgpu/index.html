Web GPU
demos

Here are a collection of simple Web GPU examples. They should work in the latest WebKit builds, and soon in a Safari Technology Preview release. The full specification is a work-in-progress on GitHub, and the implementation may differ from the current API.

Make sure you are on a system with Web GPU enabled. In Safari, first make sure the Developer Menu is visible (Preferences → Advanced), then ensure Develop → Experimental Features → Web GPU is checked.

Web GPU is an experimental technology, and you should not browse the entire Web with it enabled for now. It doesn't do much validation of content, and thus may cause some issues with your computer.
